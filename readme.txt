== Description ==

This currently exists as a pull request to Jetpack ( https://github.com/Automattic/jetpack/pull/3152 ).  For testing and folks that want to use it in the interim, I created this plugin, and re-namespaced stuff so it would still play nice, even after the feature merges into Jetpack.
